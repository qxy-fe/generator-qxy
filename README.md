## Usage

Install `yo` globally:

```shell
npm i -g yo
```

```shell
yarn global add yo
```

```shell
pnpm add yo --global
```

Install generator:

```shell
npm i -g generator-qxy
```

Use the generator:

```shell
# Run specifield generator
yo qxy:eslint
```

## Update

1. Reinstall generator globally:

```shell
npm i -g generator-qxy
```

2. Run `yo` to update generator interatively:

```shell
yo
```
