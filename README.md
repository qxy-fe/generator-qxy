## Usage

Install `yo` globally:

```sh
$ npm i -g yo
# OR
$ yarn global add yo
```

Install generator:

```sh
$ npm i -g generator-qxy
```

Use the generator:

```sh
$ yo qxy
```

## Update

Reinstall generator globally:

```
$ npm i -g generator-qxy
```

or run `yo` to update generator interatively:

```sh
$ yo
```
